# React Chrome Extension Boilerplate

A modern starter for building Chrome Extensions using React, Vite, and Manifest V3.

## Features

-   **React 18**: UI library.
-   **Vite**: Fast build tool.
-   **Manifest V3**: Compliant with latest WebExtensions standard.
-   **Hot Reloading**: Fast development experience.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Build**:
    ```bash
    npm run build
    ```
3.  **Load in Chrome**:
    -   Open Chrome and navigate to `chrome://extensions`.
    -   Enable "Developer mode".
    -   Click "Load unpacked".
    -   Select the `dist` folder generated by the build.

## Project Structure

-   `src/`: React source code.
-   `src/background.js`: Background service worker.
-   `src/content.js`: Content script injected into pages.
-   `public/manifest.json`: Extension configuration.
-   `vite.config.js`: Build configuration.
